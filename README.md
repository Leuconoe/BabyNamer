# BabyNamer

[대한민국 법원 전자가족관계시스템](https://efamily.scourt.go.kr/)에서 제공하는 출생 신고 이름 현황 통계를 이용한 아기 이름 추천 프로그램.

2008년부터 2023년까지의 최신 데이터를 기반으로 아기 이름을 추천한다.
